Como rodar essa bomba:
git clone https://github.com/lucaumxI/Trabalho_1_Distribuidos.git (clonar o repositório)
python3 -m venv .venv (cria a venv)
source .venv/bin/activate (linux) ou \.venv\Scripts\activate (windows)
pip install -r requirements.txt (instala todas as dependencias na venv)

agora só se divertir codando
